![GitHub Actions CI](https://github.com/post-luxembourg/docker-pmp/workflows/GitHub%20Actions%20CI/badge.svg)

## Setup

1. Copy and edit the db config:

```
cp postgres.env.sample postgres.env
vim postgres.env
```

2. Run

```
docker-compose up
```
